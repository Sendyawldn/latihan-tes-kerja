## 🎯 Tentang Proyek

Website latihan mandiri yang dirancang khusus untuk mahasiswa, fresh graduate, dan pencari kerja di Indonesia. Platform ini memfasilitasi persiapan menghadapi berbagai jenis tes seleksi umum yang sering digunakan oleh perusahaan (termasuk BUMN), dengan menyediakan simulasi yang mendekati kondisi nyata.

## ✨ Fitur Utama

- **⏱️ Timer Terintegrasi**<br/>
  Simulasi waktu nyata (countdown) dengan auto-submit saat durasi habis, melatih manajemen waktu dalam pengerjaan tes.
- **🔀 Pengacakan Soal (Randomized)**<br/>
  Urutan soal diacak pada setiap sesi baru untuk menghindari hafalan dan memaksimalkan efektivitas latihan.
- **📊 Sistem Penilaian Otomatis**<br/>
  Skor langsung keluar setelah tes selesai, dilengkapi dengan pembahasan singkat untuk setiap soal.
- **📈 Riwayat & Progres Latihan**<br/>
  Perekaman data skor dan waktu secara berkala untuk memantau perkembangan kesiapan.
- **📱 Responsif (Mobile-Friendly)**<br/>
  Desain dirancang agar optimal dan nyaman digunakan baik di Desktop maupun Smartphone.

## 📚 Modul Tes yang Tersedia

### 1. Simulasi Gabungan (Flagship) 🚀
Paket komprehensif berisi 100 soal dalam durasi 60 menit, mencakup 5 kategori utama:
- 🧮 **Matematika & Numerik** (20 soal)
- 🧠 **Verbal & Penalaran Logika** (25 soal)
- 👁️ **Penalaran Visual & Pola** (15 soal)
- 💻 **Pengetahuan Komputer Dasar** (20 soal)
- 🇬🇧 **Bahasa Inggris** (20 soal)

### 2. Tes Kecepatan & Ketelitian (Tes Koran) ⚡
- **Tes Kraepelin**: Simulasi menjumlahkan deret angka dari bawah ke atas dengan interval instruksi "pindah" secara otomatis. Disertai grafik kecepatan dan ketelitian di akhir tes.
- **Tes Pauli**: Mirip dengan Kraepelin, namun dengan arah pengerjaan dari atas ke bawah dan instruksi "garis".

### 3. Tes Kepribadian / SJT (Situational Judgment Test) 🧩
*(Opsional)* Simulasi penilaian kepribadian ringan untuk mengidentifikasi kecenderungan profil kandidat dalam konteks lingkungan kerja profesional.

## 🚀 Panduan Memulai (Getting Started)

Ikuti langkah-langkah berikut untuk menjalankan project ini secara lokal di mesin Anda.

### 1. Prasyarat
Pastikan Anda telah menginstal **Node.js** (versi 18.17 atau lebih baru) dan paket manajer seperti **npm**, **yarn**, atau **pnpm**.

### 2. Instalasi
Clone repository dan instal semua dependensi.

```bash
git clone https://github.com/Sendyawldn/latihan-tes-kerja.git
cd latihan-tes-kerja
npm install
```

### 3. Menjalankan Development Server
Mulai server untuk lingkungan pengembangan.

```bash
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) pada browser Anda untuk melihat hasilnya.
